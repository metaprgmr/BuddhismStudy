文字處理操作

轉Big5為UTF8

　使用中遇到個別問題，儘量修正。比如：OCR識別錯誤，等等。Big5編碼，不如UTF-8方便。

　The following works—reliably!—albeit a manual process.

  1. Make a backup copy of 9999.htm, say, into 9999big5.htm.
    Then (1) open 9999.htm in Firefox, (2) view source, and (3) copy all.
  2. Open the original 9999.htm file in vim, replace the content from above, and modify as follows:
    1. Change charset=utf-8 (from Big5 or big5);
    2. Change <div class=book> into <div class=bookClean>
    3. Look for a line that contains “Begin of Hotrank Counter”;
      if present, remove all up to </body>, and add this:
         <div class=endImage title="UTF-8 encoded"></div>
    4. Within vim, save the file with utf-8 encoding:   :w ++enc=utf-8 9999.htm
      Better exit vim right away.
    The BfnnCommon tool does the first three steps.
    Once text is pasted, just save with utf-8 in step (d), and run BfnnCommon, redirecting into a file.
